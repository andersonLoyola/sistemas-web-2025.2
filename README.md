# sistemas-web-2025.2

### Sistema de eventos de GameJam onde usuários e empresas se encontram para desenvolver jogos

#### Funcionalidades
- Criação de contas de usuários
- Papéis de usuário comum e empresa
- Autenticação
- Criação, edição e visualização de eventos
- Cadastro de informações dos jogos em um evento
- Votação/ranking de melhor jogo por evento
